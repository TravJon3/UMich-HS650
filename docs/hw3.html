1 HW Problem 3.1

For each of the probability distributions: , Uniform(1,10), Student’s T (df=1), Cauchy, Negative Binomial(10, 0.5), Chi-Square (df=10), Poisson (5),

  1. Generate the following plots functions:
    • Density
    • CDF
    • quantile (inverse-CDF)
  2. Report the first 4 moments (mean, variance, skewness, kurtosis)
  3. Complete the discrete probability distributions table below. The cell values in the table represent the values of the quantile function for the corresponding p-value (column) and distribution (row).

1.1 Weibull (1,5) Distribution

i <- seq(0, 10, 0.1)
p <- seq(0.001, 0.999, 0.001)
dfWeib <- data.frame(x=i, density=dweibull(i, 1, 5), distribution=pweibull(i, 1, 5))
qWeib <- data.frame(x=p, quantile=qweibull(p, 1, 5))

plot_ly(x= ~dfWeib$x, y= ~dfWeib$density, type="scatter", mode="lines", name="Density") %>%
  add_lines(x= ~dfWeib$x, y= ~dfWeib$distribution, name="Distribution") %>%
  layout(title="Weibull Curve", legend=list(orientation="h"), xaxis=list(title="X-Values"), yaxis=list(title="Y-Values"))
plot_ly(x= ~qWeib$x, y= ~qWeib$quantile, type="scatter", mode="lines", name="Quant") %>%
  layout(title="Weibull Quantile", legend=list(orientation="h"), xaxis=list(title="Probabilities"), yaxis=list(title="Values"))
